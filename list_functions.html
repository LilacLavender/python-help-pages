Python List Functions

Overview:

  • len()
  • max() and min()
  • index()
  • insert()

len()

The len() function is a built-in Python function that returns the length of a given object(list, dictionary, tuple, string) specified in the argument between the parentheses. The syntax for len() is len(list_name). len() returns an integer value, so you can assign it to a variable and/or manipulate it.


Example for len():

lst = ['spam', 'eggs', 'bacon'] # Defining the list
list_length = len(lst) # Storing the value len(lst) returns in a variable
print(list_length) # Outputting the value of the variable containing the list's length

Output:
>>>3

In the first line, we defined a list containing the elements 'spam', 'eggs', and 'bacon'. In the second line, we stored the value that the len() function returned in a variable called list_length. In the last line, we printed the variable list_length, which contained the length of the list. If there's any confusion, even though Python's indexing system starts at zero and the last index in the list is index 2, len() will still return 3. There's no correlation.

max() and min()

The max() and min() are both functions that can be used to find the minimum and maximum number in Python lists. They can be used in many other situations, but this is focusing on lists. The syntax for both max() and min() are realitively similar, and almost identical to len(). The syntax for max() is max(list_name), and the syntax for min() is min(list_name). max() and min() both return values, and like len() these can be manipulated and assigned to variables.


Example for min() and max():

lst = [1, 2, 3] # Defining the list
lst_min = min(lst) # Assigning the value of min() to a variable
lst_max = max(lst) # Assigning the value of max() to a variable
print(lst_min) # Printing the variable containing the min() of lst
print(lst_max) # Printing the variable containing the max() of lst

Output:
>>>1
>>>3

In the first line, we defined a list containing 3 integers(1,2, and 3). Next, we used line 3 and 4 to assign the values of min() and max() respectively. In the last lines we printed the values that we recieved from the functions, and recived the output containing the greatest and least numerical values in the list.

index()

The index() function is used to return the index of a provided value. The syntax for index() is different from min(), max(), and len() because index() is a method of Python's built-in list class. The syntax for index is list_name.index(value), where value is the value you want to locate in the list. index() returns an integer value, so you can print, assign, and manipulate the value directly from the function call.


Example for index():

lst = ['spam', 'eggs', 'bacon'] # Defining the list
eggs_index = lst.index('eggs') # Getting the index of value 'eggs' from list lst
print(eggs_index) # Printing the variable containing the index
print(lst[eggs_index]) # Printing the value at the index we got from the function

Output:
>>>1
>>>eggs

Like usual, we defined a list in line one. The second line we used to demonstrate usage of the function, by storing the return value of a function call of index() inside of a variable named eggs_index. The last two lines were to show what values we get, and that it is indeed correct. Unlike len(), the index() function and Python's index system are indeed correlated. If the value is the second element, the function would return the value 1, because it is at index 1.

insert()

Unlike all the other functions, insert() takes two arguments. One for the index you want to insert the value, and the other one is the value. The insert() function inserts a value into a list at a provided value, pushing all the elements after it(including the one at the specified index) one space to make room for the new value. The syntac for index() is list_name.insert(ind, value) where ind is the index you want to insert it at, and value is the value you want to insert. insert() does not return a value.


Example for insert():

lst = ['spam', 'bacon'] # Defining a list
print(lst) # Print the list BEFORE modification
lst.insert(1, 'eggs') # Using the insert() function to insert 'eggs' at index 1
print(lst) # Printing the list AFTER modification

Output:
>>>['spam', 'bacon']
>>>['spam', 'eggs', 'bacon']

In lines 1 and 2 we defined the list, and then printed the pre-modified list. In lines 3 and 4, we used the insert() function and printed out the list after modification. As you can see, the value 'bacon' was moved back from index 1 to index 2, to make room for 'eggs' which pushed its way into index 1. You can think of the insert() function as a person(the value) cutting in the line(the list), effectively pushing everyone behind them back a spot.
